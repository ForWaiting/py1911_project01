rest_framework 使用流程
1. 安装模块pip install djangorestframework
2.在django工程 setting注册 INSTALLED_APPS[ 'rest_framework' ]
3.在应用下方新建serializers.py 编写序列化类，需要导入模型类和from rest_framework import serializers
4.在 应用下方views 创建分类视图导入 from rest_framework import viewsets 和序列化类 编写分类视图
5，在django工程 的 urls 引入DRF在带的路由管理类 from rest_framework import routers
